# 🏛️ Museum Batak

> Platform digital interaktif untuk mengenalkan sejarah, kebudayaan, dan koleksi artefak Museum Batak kepada masyarakat luas.

---

## 📌 Tentang Proyek

Proyek **Museum Batak** ini dirancang untuk mempermudah akses informasi mengenai kekayaan budaya Batak (seperti sejarah, Rumah Bolon, kain Ulos, aksara Batak, serta berbagai artefak bersejarah) secara digital dan modern.

---

## ✨ Fitur Utama

- **📜 Galeri Koleksi & Artefak:** Informasi detail mengenai benda-benda bersejarah dan kebudayaan Batak.
- **📚 Edukasi Budaya & Sejarah:** Artikel dan materi interaktif seputar sejarah dan tradisi suku Batak.
- **📍 Informasi Kunjungan:** Panduan lokasi, jadwal operasional, dan fasilitas museum.
- **📱 Desain Responsif:** Tampilan yang nyaman diakses melalui perangkat desktop maupun smartphone.

---

## 🛠️ Teknologi yang Digunakan

* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Tailwind CSS / Bootstrap
* **Backend:** Node.js / Python / PHP (Opsional)
* **Database:** MySQL / MongoDB / PostgreSQL (Opsional)

---

## 🚀 Cara Memulai (Getting Started)

Untuk menjalankan proyek ini secara lokal di komputer kamu, ikuti langkah-langkah berikut:

### 1. Clone Repositori
```bash
git clone https://github.com/AbramSin17/museum-batak.git
cd museum-batak
```

### 2. Instalasi Dependensi
```bash
npm install
```

### 3. Jalankan Proyek
```bash
npm run dev
# atau
npm start
```
Buka browser dan akses `http://localhost:3000` (atau port yang tertera di terminal).
